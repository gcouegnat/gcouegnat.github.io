    Endommagement interlaminaire

    Le but de cette séance est d’aller vers la simulation du délaminage dans les composites stratifiés.

    Cours

    Support de cours [pdf]

    Dans un premier temps, nous verrons quelles sont les sources de contraintes hors-plan pouvant conduire au délaminage. Nous verrons ensuite comment introduire numériquement le délaminage à travers l’utilisation de modèles de zones cohésives (CZM).

    Origines des contraintes hors-plan

    • théorie des stratifiés ; hypothèse de Saint-Venant
    • bords libres, singularités
      • effet Poisson → (\sigma_{23}) et (\sigma_{33})
      • empilement (\pm\theta) → (\sigma_{13})

    Modèle de zones cohésives (CZM)

    • Principe ; formulation
    • Précautions numériques

    Exercices

    Exercice 1

    La première serie d’exercice vous propose de visualiser les états de contraintes hors-plan dus à l’effet Poisson dans le cas d’un empilement [0/90]s et au cisaillement interfacial pour un empilement [+45/-45]s.

    Enoncé des exercices [pdf]

    • Fichier *.inp : calcul1.inp

    • Maillage cas [0/90]s :

      • mesh1-twoplies-oneperply.inp
      • mesh1-twoplies-fourperply.inp
      • mesh1-twoplies-eightperply.inp

    • Maillage cas [+45/-45]s :

      • mesh2-twoplies-oneperply.inp
      • mesh2-twoplies-fourperply.inp
      • mesh2-twoplies-eightperply.inp

    Exercice 2

    Le second exercice vous propose d’utiliser des éléments CZM pour simuler un essai DCB.

    L’essai DCB est l’un des essais standards pour caractériser les propriétés interlaminaires de stratifiés composites.

    Enoncé de l’exercice [pdf]

    • Fichier *.inp : calcul-dcb.inp
    • Maillage : mesh-dcb-fine.inp
