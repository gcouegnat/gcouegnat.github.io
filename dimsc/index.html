    Matériaux et Structures Composites

    Enseignant : Guillaume Couégnat, LCTS (couegnat@lcts.u-bordeaux.fr)

    Objectif

    L’objectif de ce module est :

    • de faire le lien entre la théorie des stratifiés et sa mise en oeuvre pratique dans un code éléments finis industriel (Abaqus)
    • de vous donner un panorama des différentes approches de modélisation utilisables pour les composites stratifiés
    • de vous donner un rapide aperçu des ingrédients numériques pour la simulation de l’endommagement des composites stratifiés

    Organisation

    Le module est structuré en trois séances. Chaque séance comprend une partie “cours” d’environ 1h et une mise en pratique des notions abordées sous forme de TP.

    Les trois séances aborderont les sujet suivants :

    • Séance 1 : modélisation des composites stratifiés par des modèles coques (homogènes, généralisées, composites) ; effets de l’empilement sur le couplage membrane-flexion
    • Séance 2 : anatomie des fichiers *.inp ; modélisation de l’endommagement des plis ; modèle d’endommagement de Hashin ; simulation d’un essai de traction sur éprouvette trouée
    • Séance 3 : bords libres et contraintes hors-plan ; endommagement inter-laminaire ; modèles de zones cohésives ; simulation d’un essai DCB

    Evaluation

    L’évaluation du module se fera par un QCM à la fin de la dernière séance. Votre comportement et votre travail pendant les séances de TP seront pris en compte dans la notation finale.
