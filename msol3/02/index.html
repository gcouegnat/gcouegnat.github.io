    Support du cours [pdf]

    • Exemple d’un maillage au format Abaqus : mesh1.inp
    • Exemple d’un fichier de calcul : calcul-endo-1.inp

    Pour lancer le calcul calcul-endo-1.inp en ligne de commande, dans un terminal tapez

    abaqus interactive job=moncalcul input=calcul-endo-1

    Pour ouvrir un terminal Abaqus :

    • Sous Linux (ENSEIRB-MATMECA) : Applications / Autres / Simulia Abaqus 6.xx
    • Sous Linux (accès à distance) : voir Exécution d’Abaqus à partir du bureau à distance [pdf]
    • Sous Windows (version Student) : Démarrer / SIMULIA / Abaqus Commands
